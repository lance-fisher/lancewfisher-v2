Deployment Options
Three Tiers of Private Intelligence

Choose the deployment that matches your workflow. Every tier runs locally. Cloud providers are always opt-in, never required.

Personal Assistant
ONE MODEL, BASIC CHAT
A single local model for everyday coding questions, text generation, and quick lookups. Runs on modest hardware with no GPU required for smaller models.
● 8GB RAM minimum
● Integrated GPU (or CPU-only)
● ~5GB disk for one 7B model
● qwen2.5-coder:7b recommended
RECOMMENDED
Prosumer Workstation
MULTIPLE MODELS, CODE ASSISTANT
Multiple local models with intelligent routing. The system selects the best model per task: code models for programming, general models for reasoning, fast models for simple queries.
● 16GB RAM recommended
● RTX 3060+ or equivalent
● ~25GB disk for 3-5 models
● Auto-routing between models
Full Private Intelligence
ALL MODELS, SERVICE HEALTH, REMOTE ACCESS
Complete local AI infrastructure: multiple models, service health monitoring, session history, phone access, and optional hybrid cloud routing for tasks that benefit from larger models.
● 32GB+ RAM
● RTX 4070+ or equivalent
● ~80GB disk for full model suite
● Remote access via HTTPS + DDNS
Intelligent Routing
Why That Model Was Selected

Every routing decision is visible. The system analyzes your prompt, scores it against model capabilities, and shows you exactly why a particular model was chosen.

"Fix the TypeScript error in my auth middleware"
LOCAL qwen2.5-coder:14b
Reason: Code patterns detected, TypeScript context, local model has code specialization
"Analyze this 50-page contract for liability clauses"
CLOUD claude-opus-4-6
Reason: Long context required (200k tokens), reasoning task, cloud relay gate open
"Summarize this meeting transcript with action items"
LOCAL qwen2.5:14b-instruct
Reason: General reasoning task, sensitive content (default local), no cloud required
"Quick: what's the syntax for a Python list comprehension?"
FAST qwen2.5-coder:7b
Reason: Simple query, fast model sufficient, lower latency preferred
