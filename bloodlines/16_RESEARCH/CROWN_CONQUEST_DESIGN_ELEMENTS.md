# Crown & Conquest Design Elements

Extracted from `frontier-bastion/CROWN_AND_CONQUEST.md` for Bloodlines reference.
These RTS design concepts may inform Bloodlines dynasty strategy mechanics.

## Elevation Combat Mechanics
- Maps have multiple elevation levels (0-3)
- Higher ground grants combat advantages (damage bonus, vision range)
- Units move slower uphill, faster downhill
- Chokepoints at elevation transitions create tactical depth
- **Bloodlines application**: Dynasty strongholds on elevated terrain, siege mechanics

## Map Archetypes
- **Rush**: Small maps, close spawns, early aggression favored
- **Turtle**: Defensible positions, resource-rich interiors, late-game builds
- **Expansion**: Multiple resource nodes spread across map, control = economy
- **Bloodlines application**: Campaign map variety, each archetype favors different dynasty strategies

## Neutral Structures
- **Watchtower**: Provides vision in fog of war when captured
- **Gold Mine**: Passive income source, worth fighting over
- **Ruined Fortress**: Repairable defensive position
- **Bridge**: Chokepoint over impassable terrain
- **Bloodlines application**: Sacred sites, ancestral ruins, trade routes as contested objectives

## Resource Gathering & Economy
- Workers harvest from resource nodes (gold, wood)
- Resources spent on buildings, units, upgrades
- Economy management is core strategic layer
- Supply/population cap limits army size
- **Bloodlines application**: Dynasty wealth, tribute systems, resource allocation across generations

## Base Building Constraints
- Buildings placed on buildable terrain only
- Adjacency bonuses for related buildings
- Tech tree: certain buildings unlock others
- **Bloodlines application**: Dynasty seat construction, generational upgrades to holdings

## Unit Training & Control Groups
- Units trained from production buildings (queue system)
- Control groups for managing multiple squads
- Formation movement, attack-move, patrol orders
- **Bloodlines application**: Retinue management, heir commanding units, lineage-specific troops

## 10-Wave Survival Mode
- Escalating enemy waves with increasing difficulty
- Score tracking and performance metrics
- Timer between waves for preparation
- **Bloodlines application**: Dynasty survival scenarios, defending against rival houses
