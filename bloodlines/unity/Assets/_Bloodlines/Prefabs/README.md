# Prefabs

Approved runtime-facing prefabs belong here.

Use this lane for:

- unit prefabs
- building prefabs
- environment prefabs
- UI prefabs

Keep experiments in `Prefabs/Staging/`.

Future grouping:

- `Units/Shared`
- `Units/Dynasties/<house>`
- `Buildings/Shared`
- `Buildings/Dynasties/<house>`
- `Environment/Biomes/<biome>`
- `UI/<family>`

The first anchor folders for these groupings are now present and should be used for approved prefabs, while experiments remain in `Prefabs/Staging/`.
