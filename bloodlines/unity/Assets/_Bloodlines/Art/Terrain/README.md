# Terrain

Approved terrain art, terrain materials, and terrain-support assets belong here after staging review.

Use this lane for:

- biome material families
- transition sets
- roads and ford surfaces
- resource-ground treatments

Suggested grouping:

- `Biomes/`
- `Transitions/`
- `Roads/`
- `ResourceGround/`

These anchor folders are now present and should be used for approved terrain-facing assets.

Keep terrain-readable values and edge blending discipline aligned to the graphics lane docs.
