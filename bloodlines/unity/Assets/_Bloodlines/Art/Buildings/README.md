# Buildings

Approved runtime-facing building art belongs here.

Use this lane for:

- keeps and command halls
- civic structures
- military structures
- fortification pieces
- faith and neutral structures once integration-ready

Suggested grouping:

- `Shared/Civic`
- `Shared/Military`
- `Shared/Fortification`
- `Dynasties/<house>`
- `Faith/<covenant>`
- `Neutral`

The `Shared/`, `Dynasties/`, `Faith/`, and `Neutral/` anchor folders are now present and should receive approved structure families.

Do not place staging sheets or loose experiments here.
