# Art Staging

Temporary and reviewed art-in-progress lives here before promotion into approved runtime folders.

This folder is for:

- placeholder art
- first-pass concepts
- approved direction sheets
- in-engine test assets

It is not the final runtime destination.

## Concept Sheet Note

- `ConceptSheets/` holds the mirrored graphics-lane SVG review sheets.
- `ConceptSheetsRasterized/` holds Unity-generated PNG review boards derived from those SVG sheets.
- The project intentionally does not install the legacy external `com.unity.vectorgraphics` package. Use the Bloodlines-local raster path instead:
  - `Bloodlines -> Graphics -> Sync And Rasterize Concept Sheets`
  - `Bloodlines -> Graphics -> Rasterize Concept Sheets`
