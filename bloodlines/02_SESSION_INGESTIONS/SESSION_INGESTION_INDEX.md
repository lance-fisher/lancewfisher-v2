# Bloodlines — Session Ingestion Index

Chronological index of all design session ingestions.

## How Session Ingestion Works

1. After a design session, the raw conversation is processed using the Session Memory Ingestion Prompt
2. A session file is created in this directory: SESSION_YYYY-MM-DD_description.md
3. Key content is appended to 01_CANON/BLOODLINES_MASTER_MEMORY.md
4. Canonical rules are updated if design elements changed status
5. The session is added to this index
