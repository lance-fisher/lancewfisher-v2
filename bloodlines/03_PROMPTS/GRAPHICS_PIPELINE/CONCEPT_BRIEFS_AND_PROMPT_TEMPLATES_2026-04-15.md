# Concept Briefs And Prompt Templates

## Global Prompt Guardrail

Every Bloodlines visual prompt should preserve these negatives:

- no cartoon style
- no anime proportions
- no glossy mobile-game finish
- no exaggerated fantasy spikes
- no oversized weapons
- no floating magical ornament
- no photoreal mud overload
- no bright toy-like saturation

## Unit Prompt Template

```text
Create a grounded medieval RTS concept sheet for Bloodlines.
Asset family: [unit family]
House lane: [house or universal]
Tone: serious, weighty, dynastic medieval war game
Gameplay readability target: strong identification from RTS gameplay height
Silhouette priority: [shield / spear / bow / cavalry / command / elite / worker]
Materials: [materials]
Color use: restrained House accents only
Detail rule: clear structure and cloth hierarchy, no visual noise
Pose set: idle, march, combat-ready
Include: front, three-quarter, rear, gameplay-height read inset
Avoid: cartoon fantasy, oversized ornament, glowing magic treatment, modern fantasy glamor
```

## Building Prompt Template

```text
Create a production concept sheet for a Bloodlines RTS building.
Building family: [family]
House or universal lane: [house/universal]
Function: [economic / military / civic / faith / defensive]
Construction logic: believable load-bearing medieval structure
Readability target: identifiable at RTS gameplay height
Include: intact state, damaged state, construction-phase progression
Materials: [stone / timber / tile / plaster / iron / thatch]
Color rule: House identity through trim and banners, not full-surface paint
Avoid: fantasy excess, decorative clutter, impractical architecture
```

## Terrain Tile Prompt Template

```text
Create a Bloodlines terrain production sheet.
Biome family: [biome]
Surface type: [grass / mud / rock / frost / shoreline / forest floor]
Readability goal: pathing, elevation, and biome identity remain clear from RTS height
Include: tile variation set, transition set, edge blending examples, prop density notes
Tone: grounded, historical, post-Frost recovery world
Avoid: noisy microtexture, flat game-board look, saturated fantasy palette
```

## Resource Node Prompt Template

```text
Create a Bloodlines resource-node concept sheet.
Node type: [wood / stone / iron / water / farm support]
Gameplay role: economy and map control
Readability goal: node type readable at gameplay height
Include: base node, worked node, depleted state, local ground treatment
Avoid: generic fantasy crystal language, unclear silhouette, excessive prop clutter
```

## Biome Sheet Prompt Template

```text
Create a Bloodlines biome sheet for [biome].
Show: ground language, road language, water edge language, structure fit, prop density, lighting notes
Support: zoomed-out theatre readability and close battlefield credibility
Avoid: style drift into painterly fantasy or photoreal mush
```

## Icon Sheet Prompt Template

```text
Create a Bloodlines icon sheet.
Category: [units / buildings / resources / faith / alerts / command]
Rule: one dominant symbol per icon, clean silhouette, high legibility at small size
Palette: restrained, faction-aware only where required
Avoid: miniature illustrations, muddy gradients, noisy texture
```

## House Style Sheet Prompt Template

```text
Create a Bloodlines House style sheet for [House].
Use the approved House visual identity pack.
Show: command unit, line infantry, worker, civic building trim, military building trim, banner system, material board
Goal: immediate battlefield recognition without cartoon exaggeration
Avoid: confusing overlap with [confusion risk houses]
```
