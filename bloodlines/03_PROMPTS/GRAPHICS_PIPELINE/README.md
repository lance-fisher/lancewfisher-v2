# Graphics Prompt Framework

This folder contains the reusable visual-production prompt and briefing system for Bloodlines.

It is connected to:

- `13_AUDIO_VISUAL/GRAPHICS_PIPELINE/`
- `14_ASSETS/GRAPHICS_PIPELINE/`
- `docs/unity/VISUAL_ASSET_PIPELINE_2026-04-15.md`
- `docs/unity/BLOODLINES_UNITY_6_3_VISUAL_IMPLEMENTATION_GUIDE_2026-04-16.md`

Use these files when generating:

- concept sheets
- style sheets
- icon sheets
- placeholder briefs
- House-specific asset batches

Do not generate art from memory alone. Use the House identity packs and manifest notes first.
