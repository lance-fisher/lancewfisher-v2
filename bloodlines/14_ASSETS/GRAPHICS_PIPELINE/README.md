# Graphics Asset Staging

This folder is the non-destructive asset staging lane for Bloodlines visual production.

## Stage Directories

- `00_INBOX`
- `01_PLACEHOLDER`
- `02_FIRST_PASS_CONCEPT`
- `03_APPROVED_DIRECTION`
- `04_PRODUCTION_CANDIDATE`
- `05_IN_ENGINE_TEST`
- `06_REFINEMENT_CANDIDATE`
- `07_NEAR_FINAL`
- `08_FINAL`
- `99_LEGACY_REVIEW_HOLD`

## Use

- Put raw incoming concepts or outside-safe imports in `00_INBOX`.
- Move only reviewed assets forward.
- Use `99_LEGACY_REVIEW_HOLD` instead of deleting uncertain material.
- Preserve provenance and naming whenever possible.

## Current Audit Result

At lane creation time, the active canonical root had no governed production art assets here yet. This folder structure exists so future sessions can start real asset production without mixing raw concepts into unrelated design folders.
